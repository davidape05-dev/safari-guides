# 🦁 KenyaGuide - Tour Guide Discovery Platform

A web-based system connecting tourists with professional tour guides in Kenya.

## 📋 Project Overview

SafariGuide is a centralized platform where tourists can discover, evaluate, and book professional tour guides based on their specific interests. The system addresses the gap between tourists seeking authentic Kenyan experiences and qualified guides looking to showcase their expertise.

## ✨ Features

### For Tourists
- Search and filter guides by location, price, category, and rating
- View detailed guide profiles with portfolios
- Book tours with date selection
- Make secure payments via M-Pesa/Card
- Leave reviews and ratings after completed tours

### For Guides
- Create professional profiles with portfolios
- Manage booking requests (accept/decline)
- Track earnings and request payouts
- View reviews from tourists

### For Admin
- Verify guide applications
- Manage users and bookings
- Process payments
- Generate system reports

## 🗄️ Database Schema

The database consists of 10+ tables including:
- `users` - User accounts (tourist, guide, admin)
- `guide_profiles` - Extended guide information
- `bookings` - Tour bookings
- `payments` - Payment transactions
- `reviews` - Tourist reviews and ratings
- `notifications` - System notifications
